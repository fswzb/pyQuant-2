# easyquotation

### 前言
* 用于获取新浪的免费实时行情
* 有兴趣的可以加群 `429011814` 一起讨论

**开发环境** : `Ubuntu 15.10` / `Python 3.5`

### requirements

> Python 3.5+
 
> pip install -r requirements.txt

### 用法

#### 引入:

```python
import easyquotation
```

#### 选择 sina 行情

```python
quotation = esayquotation.use('sina')
```

#### 获取所有股票行情

```python
quotation.all
```
